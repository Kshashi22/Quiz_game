# Quiz Web App

A responsive and interactive Quiz Web Application built using **HTML**, **CSS**, and **JavaScript**. The application presents multiple-choice questions, tracks the user's progress, calculates the final score, and provides a clean user experience.

---

## Features

- Multiple-choice quiz interface
- Progress bar to track quiz completion
- Automatic score calculation
- Responsive and user-friendly design
- Input validation before moving to the next question
- Option to restart the quiz after completion

---

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)

## Getting Started

### Clone the Repository

```bash
git clone https://github.com/Kshashi22/Quiz-Web-App.git
```

### Navigate to the Project Directory

```bash
cd Quiz-Web-App
```

### Run the Application

Open the `index.html` file in your preferred web browser.

For a better development experience, you can use the **Live Server** extension in Visual Studio Code.

---

## Application Flow

1. Launch the application.
2. Read the displayed question.
3. Select one of the available options.
4. Click **Next** to proceed.
5. Repeat until all questions are completed.
6. View your final score.
7. Restart the quiz if desired.
